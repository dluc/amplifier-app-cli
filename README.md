# Amplifier TUI

TUI interface for Amplifier.

## Installation

### For Users

```bash
# Install globally
uv tool install git+https://github.com/dluc/amplifier-app-cli@tui
```

## Quick Start

```bash
# First-time setup (auto-runs if no config)
# Tip: Set environment variables for faster setup
# export ANTHROPIC_API_KEY="your-key"
# The wizard detects env vars and shows them as defaults
amplifiertui init

# Run
amplifiertui
```
